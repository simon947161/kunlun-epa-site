# Repository Relationship

## Repository Role

`kunlun-epa-site` is a public-facing repository for Kunlun ecological planning, dryland resilience and long-term sustainability communication.

It supports knowledge sharing, public explanation and future partner-facing materials.

## Position in the Wider System

```text
Dryland Resilience and Ecological Planning
        ↓
Kunlun EPA Site
        ↓
Public communication and cooperation materials
```

## Relationship to Other Repositories

- `Simon` is the personal headquarters and project index.
- `civilization-dryland` provides dryland civilization and climate adaptation knowledge.
- `eco-agent-system` may provide environmental evidence and decision-support patterns.
- `climateos-knowledge-garden` stores internal CRP, research notes and project memory.
- `climateos-book` may publish broader mature concepts.

## Boundary

This repository should focus on:

- Kunlun ecological planning
- dryland resilience
- sustainability concepts
- environmental cooperation materials
- public communication
- regional development narratives

Do not use this repository as the main home for:

- private project records
- raw Obsidian notes
- generic BuildingOS standards
- multi-agent runtime design
- unrelated personal career materials

## Operating Rule

Kunlun EPA Site is a public-facing project room.

Keep it suitable for ecological planning communication and future collaboration.
